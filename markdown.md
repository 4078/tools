# Markdown

  本教程将采取**内容与形式一体化**的方式进行。具体来说，正文分为多个自然段，**奇数自然段**为实际显示内容，其紧邻的**偶数自然段**为奇数自然段内容对应的[Markdown](https://en.wikipedia.org/wiki/Markdown)形式。偶数段可为Markdown代码，因此以`代码形式`显示。
