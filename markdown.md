# Markdown

一般来说，我们学习[Markdown](https://en.wikipedia.org/wiki/Markdown)主要目的是要进行**文档与书籍写作**，因此可通过看到一本书的实际显示内容与相应的Markdown部分，即可基本达到目的。因此本教程将采取**内容与形式一体化**的方式进行。具体来说，正文分为多个自然段，**奇数自然段**为实际显示内容，其紧邻的**偶数自然段**为奇数自然段内容对应的Markdown形式。偶数段可为Markdown代码，因此以`代码形式`显示。这样，把偶数自然段的内容直接拷贝到支持Markdown的编辑环境中，即可显示对应奇数自然段。
